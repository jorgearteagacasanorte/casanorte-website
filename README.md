# Casa Norte Website Demo

Static website built with plain HTML, CSS and JavaScript for deployment on GitHub Pages.

## Preview
Open `index.html` in a browser.

## Before publishing
1. Replace the Formspree placeholder in the contact form.
2. Confirm email, phone, address and service areas.
3. Replace demo stock photos with Casa Norte-owned or licensed photography.
4. Review the privacy policy.
5. Add the final custom domain in GitHub Pages.
6. Submit `sitemap.xml` to Google Search Console.
7. Validate JSON-LD with Google's Rich Results Test.

## GEO / AI discoverability
The site uses:
- Semantic HTML and descriptive headings
- Direct, factual service descriptions
- FAQ content with FAQPage structured data
- LocalBusiness and Organization-style structured data
- Crawlable server-rendered HTML
- A consistent business name, services and service areas

`llms.txt` is included as an optional summary, but it is not a substitute for SEO and is not used by Google as a special ranking file.
